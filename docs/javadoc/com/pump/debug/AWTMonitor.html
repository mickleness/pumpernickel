com.pump.debug

Class AWTMonitor

  • java.lang.Object
    • com.pump.debug.AWTMonitor

  • public class AWTMonitor
extends java.lang.Object
    This class monitors the event dispatch thread.

    After 2 seconds if the thread is unresponsive, a list of stack traces is printed to the console.

    After 15 seconds if the thread is unresponsive, the panicListener is notified (if it is non-null). By default the panicListener is a PanicDialogPrompt, but you can nullify this value or create your own AWTPanicListener.

        public static void installAWTListener(java.lang.String applicationName,
                                      boolean addPanicDialogPrompt)
        This installs a thread that monitors the AWT thread.

        If the AWT thread is unresponsive for over 2 seconds, then thread stacks are dumped to the console. If the event dispatch thread is unresponsive for over 15 seconds, the panicListener (if non-null) is notified.

        Parameters:
        applicationName - the name of the application to present to the user
        addPanicDialogPrompt - if true then the PanicDialogPrompt is added.
