Class PaddingInfo

  • java.lang.Object
    • com.pump.awt.PaddingInfo
  • Direct Known Subclasses:
    StoredPaddingInfo

    public class PaddingInfo
extends java.lang.Object
    This retrieves information about the empty pixels in a JComponent. This class paints the component in an offscreen image and measures the empty pixels.

    Other subclasses may implement a more efficient means of determining this padding. (See StoredPaddingInfo.)
