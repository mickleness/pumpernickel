Class AnimatedLayout

  • java.lang.Object
    • com.pump.awt.AnimatedLayout
  • All Implemented Interfaces:
    java.awt.LayoutManager, java.awt.LayoutManager2
    Direct Known Subclasses:
    AnimatedLayout.ClientProperty

    public abstract class AnimatedLayout
extends java.lang.Object
implements java.awt.LayoutManager2
    A layout that smoothly animates components as they need to move from one location to another.

    Each animated movement is proportional to the distance that the component has to cover, giving the impression of a decelerating animation as the object approaches its destination.
